# CPI_Inflation_Analysis

Project Overview

This project analyzes India’s Consumer Price Index (CPI) inflation data using Microsoft Excel.
The goal is to understand inflation trends over time and present insights in a clear, visual format.

⸻

Dataset
	•	CPI inflation data for India
	•	Data includes time-based inflation values (monthly / yearly)
	•	Source: Publicly available government data

⸻

Tools Used
	•	Microsoft Excel
	•	Pivot Tables
	•	Charts & Graphs
	•	Excel formulas for analysis

⸻

Key Analysis Performed
	•	Trend analysis of CPI inflation over time
	•	Comparison of inflation across different periods
	•	Identification of inflation rise and fall patterns
	•	Visualization using charts for easy interpretation

⸻

 Key Insights
	•	Inflation shows clear fluctuations over different time periods
	•	Certain periods reflect higher inflation trends compared to others
	•	Visual analysis helps quickly understand macroeconomic movement

⸻

 Files in This Repository
	•	CPI_Inflation_Analysis.xlsx – Excel file containing data, pivots, and charts
	•	README.md – Project explanation

⸻

 Skills Demonstrated
	•	Data analysis using Excel
	•	Data cleaning and structuring
	•	Trend identification
	•	Data visualization
	•	Analytical thinking

⸻

 How This Project Is Useful
